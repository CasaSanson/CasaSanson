# Fuentes

Para usar la fuente Kugile Regular, coloca los siguientes archivos en esta carpeta:

- `Kugile-Regular.woff2` (formato recomendado para navegadores modernos)
- `Kugile-Regular.woff` (formato de respaldo)
- `Kugile-Regular.ttf` (formato de respaldo)

## Cómo obtener la fuente:

1. Descarga la fuente Kugile Regular desde el sitio oficial o una fuente confiable
2. Convierte la fuente a los formatos necesarios usando herramientas como:
   - [Font Squirrel Webfont Generator](https://www.fontsquirrel.com/tools/webfont-generator)
   - [Convertio](https://convertio.co/ttf-woff/)
3. Coloca los archivos en esta carpeta
4. La fuente se aplicará automáticamente a toda la página

## Formatos soportados:
- **WOFF2**: Mejor compresión, navegadores modernos
- **WOFF**: Buena compresión, amplio soporte
- **TTF**: Formato original, máxima compatibilidad
